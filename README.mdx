# Functional Requirements

- [x] User should be able to create a new transaction;
- [x] User should be able to list every transaction created;
- [x] User should be able to list a single transaction;
- [x] User should be able to delet a transaction;
- [x] User should be able to edit a transaction;
- [x] User should be able to list transactions summary;

# Business Rule

- [x] A transaction can be of type Income which will add to total value or Outcome which will subtract;
- [x] It should be possible to identify user between transactions;
- [x] User can only see transactions that he created;
